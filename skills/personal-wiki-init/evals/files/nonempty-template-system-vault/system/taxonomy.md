# 页面分类法

## 来源层
- `raw/inbox/`：待处理原始资料。
- `raw/library/`：正式原始资料库。
- `wiki/sources/`：与 raw 来源对应的结构化来源页。

## 知识层
- `wiki/topics/`：跨来源主题。
- `wiki/people/`：人物。
- `wiki/concepts/`：概念。
- `wiki/projects/`：项目。
- `wiki/questions/`：长期值得追踪的问题。
- `wiki/syntheses/`：综合分析。
- `wiki/timelines/`：时间线。

## 维护原则
- 优先更新已有页面。
- 只有值得长期追踪的实体才单独建页。
- 信息不足时先记录在来源页，不强行拆页。
