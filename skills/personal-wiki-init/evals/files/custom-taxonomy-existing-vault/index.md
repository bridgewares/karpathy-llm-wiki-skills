# 个人 LLM Wiki 导航

## 当前重点主题入口
- 先在 `wiki/topics/` 中建立跨来源主题页。
- 访谈类来源可优先围绕 recurring themes 建立主题与问题页。

## Raw 管理入口
- `raw/inbox/`：待处理来源入口。
- `raw/library/`：正式原始资料库。
- `raw-index.md`：唯一官方 ingest 状态表。
- `inbox.md`：记录待处理来源、待整理问题与待执行动作。

## 来源页目录入口
- `wiki/sources/interviews/`
- `wiki/sources/articles/`
- `wiki/sources/videos/`
- `wiki/sources/books/`
- `wiki/sources/podcasts/`
- `wiki/sources/webclips/`

## 知识页目录入口
- `wiki/topics/`
- `wiki/people/`
- `wiki/concepts/`
- `wiki/projects/`
- `wiki/questions/`
- `wiki/syntheses/`
- `wiki/timelines/`

## 最近新增入口
- 初始化完成后，优先把第一个真实来源放入 `raw/inbox/<type>/`。
- 首次 ingest 后，把新来源页与相关 topic/question 页面链接回来。
