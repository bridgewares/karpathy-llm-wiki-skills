# 分类法

## raw taxonomy
当前 vault 已有稳定的自定义 raw taxonomy：
- `interviews/`

补齐骨架时的原则：
- 保留 `raw/inbox/interviews/` 与 `raw/library/interviews/`。
- 保留与 `interviews` 对应的 `wiki/sources/interviews/`。
- 可以补充默认分类目录作为未来扩展，但不得要求迁移已有内容。

## wiki taxonomy
- `wiki/sources/`：按来源类型组织来源页。
- `wiki/topics/`：跨来源主题。
- `wiki/people/`：关键人物。
- `wiki/concepts/`：反复出现的概念。
- `wiki/projects/`：持续跟踪的项目。
- `wiki/questions/`：需要长期维护的问题。
- `wiki/syntheses/`：跨来源综合分析。
- `wiki/timelines/`：时间线整理。
